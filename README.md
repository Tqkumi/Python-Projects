# SLAP GAME 

## Overview
OpenCVとMediaPipeを用いたジェスチャー認識ゲームです．
WEBカメラから取得した映像を利用し，手の動きを検出してゲームを操作します．

## Technologies
- Python
- OpenCV
- MediaPipe Hand Landmarker
- Numpy

## Features
-手の位置検出
-レベルシステム
-ジェスチャーによるゲーム操作
リアルタイム映像処理

## 遊び方
1.Sキーを押してゲームを開始します．
2.カメラが起動するので，画面にある緑の線より右側に手が映るように手を置いてください
  (なるべく画面左側には手以外の顔や腕などの肌を映さないでください．誤認識してしまう可能性があります．)
3.３秒のカウントダウン後，ゲームが開始されます．
4.画面中央左に(o_o)の顔文字が表示され，１０秒のタイマーの内どこかで顔文字が赤くなりよそ見をするので，
　そのタイミングで手がその顔にかかるようにビンタしてください．
5.よそ見している間(赤く光っている間)にビンタを成功させると，次のレベルに進みます．
　失敗したらRキーでリトライ，もしくはQキーでゲームを終了してください．
6.レベル５をクリアするとゲームクリアです．リザルトが表示されるのでQキーでゲームを終了してください．
